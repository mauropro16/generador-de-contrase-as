# generador-de-contrase-as

Nombre del proyecto:
Generador seguro de contraseñas 

Integrantes:
Henry Mauricio Caizaguano Montero

Objetivo del sistema:
Desarrollar un programa que permita generar contraseñas seguras de forma aleatoria con una longitud entre 5 y 10 caracteres, utilizando letras mayúsculas, minúsculas, números y símbolos, con el fin de mejorar la seguridad de las cuentas de los usuarios.

Descripción de funcionalidades:

Mostrar un menú con las opciones de generar una contraseña o salir del programa.
Solicitar al usuario la longitud de la contraseña (entre 5 y 10 caracteres).
Validar que la longitud ingresada sea correcta.
Generar una contraseña aleatoria utilizando letras, números y caracteres especiales.
Mostrar la contraseña generada en pantalla.
Permitir generar varias contraseñas sin cerrar el programa.
Finalizar la ejecución cuando el usuario seleccione la opción de salir.

Fecha:
28 de junio de 2026


**Introduccion.**
En la actualidad existe un gran problema en cuanto a la seguridad de la información, debido a que la mayor parte de la población utiliza los medios digitales.
Por tal motivo es importante tener una gran seguridad al momento de proteger las cuentas personales.
Es por eso que se desarrolló un programa llamado "Generador Seguro de Contraseñas", con el cual el usuario genera una contraseña aleatoria con letras mayúsculas o minúsculas, números y caracteres, el cual va a ser difícil de descifrar por los ladrones informáticos. Este proyecto se desarrolló con los conocimientos adquiridos durante la asignatura de Lógica de la Programación, fortaleciendo el análisis, el diseño y el desarrollo del software.

**Descripción del Problema.**

Muchos usuarios utilizan contraseñas demasiado fáciles de recordar, por lo que son vulnerables a un ataque informático. Además, el crear una contraseña segura de manera manual puede resultar complicado para algunos usuarios. El proyecto busca facilitar la generación de contraseñas seguras, permitiendo al usuario generar una clave aleatoria con una longitud de 5 a 10 caracteres y así puedan tener una mayor protección de su información.

**Relación con los Contenidos de la Asignatura.**

El desarrollo del proyecto permitió aplicar diversos contenidos estudiados durante la asignatura. Se utilizaron algoritmos para definir la lógica del programa, diagramas de flujo y diagramas de arquitectura para representar el funcionamiento del sistema, variables para almacenar información, estructuras condicionales para validar los datos ingresados por el usuario y bucles para controlar la repetición de procesos.

**Explicación del Sistema Desarrollado.**

En este sistema se encuentran dos opciones: una para salir del programa y otra opción que nos permitirá generar la contraseña, primero preguntándonos la longitud que deseamos para la contraseña. Posterior, de acuerdo a la longitud, se eligen aleatoriamente entre letras, números y caracteres especiales. Finalmente, la contraseña generada por el sistema se muestra al usuario para que pueda utilizarla.

Programas como el generador de contraseñas ayudan a reducir los riesgos de un ataque o vulnerabilidad de las cuentas del usuario, fortaleciendo la ciberseguridad.
